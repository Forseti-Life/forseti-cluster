# Contributing

Thank you for interest in contributing to Forseti Cluster!

## Getting Started

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests
5. Commit with clear messages
6. Open a Pull Request

## Code Standards

- Follow Drupal 10 coding standards
- Write PHPDoc comments for all public methods
- Add test coverage for new functionality
- Update README for new features
